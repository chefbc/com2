# com2

## Assets and media

- **Branding and hero:** Header logo, hero image, and hero background are set in `mkdocs.yml` under `extra.branding`. Assets live in `docs/assets/` (e.g. Speckless logos, `Speckless_bg1_overlay.jpg`).
- **Notification video:** Optional background video is configured in `docs/notification.md` (`video: assets/stats-background.mp4`). Replace the file in `docs/assets/` and update the path in that file if needed.
- **Stock photos/video:** For additional or replacement imagery (hero, gallery, notification), use [Pexels](https://www.pexels.com) (e.g. search "car detailing", "car wash", "luxury car"). Download, place in `docs/assets/`, and set the path in `mkdocs.yml` or the relevant page. Pexels content is free for commercial use; add attribution in footer or credits if required by the license.
